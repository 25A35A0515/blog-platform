📝 Blog Platform with Comments

This is a full-stack blog platform where users can register, login, create blog posts, and add comments. It is built using MERN-style architecture with REST APIs and database integration.

The system allows authenticated users to manage blog posts (create, edit, delete). Other users can view posts and participate in discussions through comments.

This project helps in understanding full-stack development, authentication, and CRUD operations with real-world user interaction features.

🚀 Features
👤 User Features
User Registration
User Login (JWT Authentication)
View Blog Posts
Add Comments on Posts
✍️ Post Features
Create Blog Post
Edit Blog Post
Delete Blog Post
View All Posts
🏗 Tech Stack
Frontend: React.js (or any UI framework)
Backend: Node.js, Express.js
Database: MongoDB / MySQL (based on your project)
Authentication: JWT, bcrypt
API Type: RESTful APIs
